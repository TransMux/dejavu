# DejaVu

[中文](README_zh_CN.md)

## 💡 简介

[DejaVu](https://github.com/siyuan-note/dejavu) 是一个用于数据快照和同步的 golang 库。

## ✨ 特性

* 类似 Git 的版本控制
* 支持打标签和从标签迁出
* 文件分块去重
* AES 加密
* 连接云端存储

## ⚠️ 注意

* 不支持文件夹
* 不支持权限属性
* 不支持符号链接
